# 端口使用文档

记录项目中所有服务占用的端口。

---

## 端口分配

| 端口 | 服务 | 说明 | 状态 |
|------|------|------|------|
| 3306 | MySQL | 数据库服务 | ✅ 已启用 |
| 5000 | COS API | Flask 对象存储接口 | ✅ 已启用 |
| 5173 | Frontend | Vue 前端开发服务器 | ✅ 已启用 |
| 6379 | Redis | 缓存服务 | ✅ 已启用 |
| 8000 | User Platform | 用户管理平台 API | ✅ 已启用 |
| 8001 | Resume API | 简历附件管理服务 | ✅ 已启用 |
| 8080 | Frontend | 前端开发服务器（预留） | ⏳ 预留 |
| 15508 | Portal | Web 中转平台 | ✅ 已启用 |

---

## 端口范围规划

| 范围 | 用途 |
|------|------|
| 3000-3999 | 前端/静态服务 |
| 5000-5999 | Python/Flask 服务 |
| 6000-6999 | 缓存/消息队列 |
| 8000-8999 | Web 服务器/代理 |
| 15000-15999 | 中转平台/其他服务 |

---

## 添加新端口

新增服务时，请：
1. 选择未被占用的端口
2. 在此文档中注册
3. 更新 `CLAUDE.md` 中的端口索引
